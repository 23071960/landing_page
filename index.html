Lei Geral de Proteção de Dados (LGPD)

A Lei Geral de Proteção de Dados (LGPD) é a legislação brasileira que regula as atividades de tratamento de dados pessoais. Ela tem como objetivo proteger os direitos fundamentais de liberdade e de privacidade, bem como o livre desenvolvimento da personalidade da pessoa natural.

Principais pontos da LGPD:

  • Transparência: As organizações devem fornecer informações claras sobre a coleta, uso e armazenamento dos dados pessoais.
  • Consentimento: O consentimento do titular deve ser obtido para o tratamento de dados, salvo exceções previstas na lei.
  • Finalidade: Os dados devem ser coletados para propósitos específicos, explícitos e legítimos.
  • Segurança: As organizações devem garantir a proteção dos dados pessoais contra acessos não autorizados e situações acidentais ou ilícitas.
  • Direitos dos Titulares: Os titulares dos dados têm o direito de acessar, corrigir, eliminar e portabilizar seus dados pessoais.

Para mais informações, visite o site oficial da LGPD.
